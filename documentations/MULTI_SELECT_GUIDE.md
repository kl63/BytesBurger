## 💡 How It Works

### **Selection:**
1. **Click once** → Add to order (orange)
2. **Click again** → Remove from order (gray)
3. **Select multiple** → All stay active!

### **Price Calculation:**
- Each selected premium option adds to total
- FREE options add 0 to price
- All options add their calories
- Total updates **instantly**

### **Example Flow:**
```
Base Price: $9.99

+ Bacon (+$1.50)
+ Avocado (+$1.00)
+ Truffle Aioli (+$1.00)
─────────────────────
Item Total: $13.49

× Quantity: 2
═════════════════════
Cart Total: $26.98
```
